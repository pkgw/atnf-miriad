Miriad is a radio interferometry data reduction package. It has particular emphasis on aspects of interest to users of the Australia Telescope Compact Array (ATCA). Miriad can be used for the reduction of continuum and spectral line experiments from the original loading of the data (either FITS, the ATNF RPFITS or old BIMA formats) through to the image analysis and display stages (publication quality output).

Miriad, in particular, supports a number of niche areas. These include calibration and analysis of polarimetric data from the ATCA, multi-frequency synthesis imaging, mosaicing, ATCA pulsar bin mode, and some spectral line observing applications (e.g. Zeeman experiments).
