Miriad tutorials

The following is a list of tutorials and demonstrations that can be used to gain some understanding of Miriad, ATCA data and general radio interferometry principles.

Note: Some of these tutorial packages are quiet large. Also note that, as the tutorials have generally been developed for exercises within a workshop, they generally instruct you to unpack the data into a particular directory. These instructions can be ignored.

The tutorial packages are in the form of tarred gziped files.
